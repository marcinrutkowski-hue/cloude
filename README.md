# cloude

Tracker wpisów blogowych aplikacja.pl

## blog-tracker.csv

| Kolumna | Opis |
|---|---|
| Tytuł | Tytuł wpisu |
| Opis (o czym) | Krótki opis tematu |
| Słowo kluczowe główne | Główny target keyword |
| Słowa kluczowe wspierające | Dodatkowe KW (przecinek) |
| Status | `opublikowany` / `w kolejce` |
| Data publikacji | Format YYYY-MM-DD |
| Liczba słów | Łączna liczba słów w artykule |
| Nasycenie KW głównego (%) | % wystąpień głównego KW relative do całości |
| Nasycenie KW wspierających (%) | Średnie nasycenie KW wspierających |
| Uwagi | Dowolne notatki |
