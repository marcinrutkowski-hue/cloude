# cloude

Tracker wpisów blogowych aplikacja.pl

## blog-tracker.csv

| Kolumna | Opis |
|---|---|
| Tytuł | Tytuł wpisu |
| Opis | Krótki opis tematu |
| Keyword główne | Główny target keyword |
| Keyword wspierające | Dodatkowe keywords (średnik) |
| Status | `opublikowany` / `w kolejce` |
| Data publ. | Format YYYY-MM-DD |
| Słów | Łączna liczba słów w artykule |
| Znaki | Liczba znaków ze spacjami |
| Nas. KW gł. (%) | Nasycenie głównego keyword — patrz skala |
| Nas. KW wsp. (%) | Nasycenie keyword wspierających — patrz skala |
| Uwagi | Rekomendacje i notatki |

## Skala nasycenia słów kluczowych

| Emoji | Zakres | Ocena |
|---|---|---|
| ✅ | 0.5–2.0% | Optymalne |
| ⚠️ | 2.0–3.0% | Akceptowalne, obserwować |
| ❌ | > 3.0% lub < 0.5% | Wymaga korekty |
